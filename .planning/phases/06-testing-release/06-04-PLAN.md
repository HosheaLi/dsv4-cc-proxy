|----------|-------------|
| git tag -> CI | tag push 触发 CI publish job, 包含发布凭据访问 |
| version bump -> prod | 版本号变更影响所有下游（pip install, Docker, GitHub Release）|

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-09 | Spoofing | git tag v2.0.0 | mitigate | tag 在 main 分支本地创建后推送, 确保只有授权推送能创建 release |
| T-06-10 | Tampering | _version.py | mitigate | 版本号单源（_version.py 唯一来源）, 与 pyproject.toml 动态引用, 无二次硬编码 |
| T-06-11 | Information Disclosure | CI publish job logs | mitigate | PyPI OIDC 无需 API token, 日志不暴露敏感信息。Docker Secret 由 GitHub Actions 自动脱敏 |
</threat_model>
