    === 内部助手函数清单 ===

    1. `_build_sse_event(event_type: str, data: dict) -> str`
       - 构建完整 SSE 事件字符串: `f"event: {event_type}\ndata: {json.dumps(data, ensure_ascii=False)}\n\n"`
       - 这是所有 SSE 事件构建的底层工具（D-12）

    2. `_build_response_created(response_id: str, model: str) -> str`
       - payload: `{"type":"response.created","response":{"id":{response_id},"object":"response","created":{int(time.time())},"model":{model},"status":"in_progress","usage":null}}`

    3. `_build_response_in_progress(response_id: str) -> str`
       - payload: `{"type":"response.in_progress","response":{"id":{response_id},"object":"response","status":"in_progress"}}`

    4. `_build_output_item_added(output_index: int, item_type: str, **fields) -> str`
       - 构建通用的 output_item.added
       - item 基础结构: `{"id":, "type":, "status": "in_progress"}`
       - item_type="reasoning" -> 额外 key: `summary: []`
       - item_type="message" -> 额外 key: `role: "assistant", content: []`
       - item_type="function_call" -> 额外 key: `name: fields["name"], call_id: fields["call_id"], arguments: ""`
       - payload: `{"type":"response.output_item.added","item":{item},"output_index":{output_index}}`

    5. `_build_content_part_added(item_id: str, output_index: int, content_index: int) -> str`
       - content_index 固定为 0（每 item 只一个 content part）
       - payload: `{"type":"response.content_part.added","item_id":{item_id},"output_index":{output_index},"content_index":0,"part":{"type":"output_text","text":""}}`

    6. `_build_reasoning_text_delta(item_id: str, output_index: int, delta: str, sequence: int) -> str`
       - payload: `{"type":"response.reasoning_text.delta","item_id":{item_id},"output_index":{output_index},"content_index":0,"delta":{delta},"sequence_number":{sequence}}`

    7. `_build_output_text_delta(item_id: str, output_index: int, delta: str, sequence: int) -> str`
       - payload: `{"type":"response.output_text.delta","item_id":{item_id},"output_index":{output_index},"content_index":0,"delta":{delta},"sequence_number":{sequence}}`

    8. `_build_function_call_arguments_delta(item_id: str, call_id: str, output_index: int, delta: str, sequence: int) -> str`
       - payload: `{"type":"response.function_call_arguments.delta","item_id":{item_id},"call_id":{call_id},"output_index":{output_index},"delta":{delta},"sequence_number":{sequence}}`

    9. `_build_function_call_arguments_done(item_id: str, call_id: str, output_index: int, delta: str) -> str`
       - payload: `{"type":"response.function_call_arguments.done","item_id":{item_id},"call_id":{call_id},"output_index":{output_index},"delta":{delta}}`

    10. `_build_output_item_done(output_index: int, item_type: str, item_id: str, **fields) -> str`
        - 构建通用的 output_item.done
        - item_type="reasoning" -> item: `{"id":{item_id},"type":"reasoning","status":"completed"}`
        - item_type="message" -> item: `{"id":{item_id},"type":"message","status":"completed","content":[{"type":"output_text","text":{fields["accumulated_text"]}}]}`
        - item_type="function_call" -> item: `{"id":{item_id},"type":"function_call","name":{fields["name"]},"call_id":{fields["call_id"]},"arguments":{fields["accumulated_arguments"]},"status":"completed"}`
        - payload: `{"type":"response.output_item.done","item":{item},"output_index":{output_index}}`

    11. `_build_response_completed(response_id: str, usage: dict | None) -> str`
        - payload: `{"type":"response.completed","response":{"id":{response_id},"object":"response","status":"completed","usage":{usage or {"input_tokens":0,"output_tokens":0,"total_tokens":0}}}}`

    === 主函数：translate_sse_stream ===

    `async def translate_sse_stream(upstream: AsyncIterable[dict]) -> AsyncGenerator[str, None]:`

    接收已解码的 Chat delta dict 的异步迭代器，yield Responses API SSE 事件字符串。

    **状态变量（全部为局部变量，D-06 隐式状态）：**
    ```python
    current_output_type: str | None = None  # None | "reasoning" | "text" | "tool_call"
    active_tool_indices: set[int] = set()   # D-07: 追踪活跃 tool_call indices
    is_first_chunk = True
    _completed = False                       # D-08: finish_reason 幂等保护
    sequence = 0                             # 递增序列号
    response_id = f"resp_{uuid4().hex[:12]}"
    item_id_counter = 0                      # 递增 item_id
    output_counter = 0                       # 递增 output_index

    # 工具调用追踪（每个 index 独立）
    tool_call_ids: dict[int, str] = {}       # index -> call_id (如 "call_1")
    tool_call_names: dict[int, str] = {}     # index -> name (如 "get_weather")
    tool_call_arguments: dict[int, str] = {} # index -> 累计 arguments 字符串

    # 文本/推理内容累计（用于 output_item.done）
    accumulated_text = ""
    accumulated_reasoning = ""
    ```

    **处理逻辑（严格遵循 D-08/D-09 生命周期）：**

    1. `async for chunk in upstream:` - 外层循环
       - `choices = chunk.get("choices", [])` -> 空则 `continue`
       - `delta = choices[0].get("delta", {})`
       - `finish_reason = choices[0].get("finish_reason")`
       - `if _completed: continue` (Pitfall 4 幂等保护)

    2. 首个 chunk（`is_first_chunk` 判断）：
       - yield `_build_response_created(response_id, {model from chunk or "deepseek-v4-pro"})`
       - yield `_build_response_in_progress(response_id)`
       - `is_first_chunk = False`
       - （获取 initial model name: `chunk.get("model", "deepseek-v4-pro")`）

    3. **`delta.reasoning_content` 处理（D-08 首个 reasoning_content）：**
       - `reasoning = delta.get("reasoning_content")`
       - 若为似真值：
         - `current_output_type == "reasoning"` 时不做 transition（增量延续）
         - `current_output_type != "reasoning"` 且 `current_output_type is not None`：
           - yield `_build_output_item_done(output_counter, current_output_type, item_id, ...)`
         - `current_output_type != "reasoning"` 时：
           - `item_id_counter += 1` -> 新 item `f"item_{item_id_counter}"`
           - `output_counter += 1`
           - yield `_build_output_item_added(output_counter, "reasoning", ...)`
           - `current_output_type = "reasoning"`
         - `sequence += 1`
         - `accumulated_reasoning += reasoning`
         - yield `_build_reasoning_text_delta(item_id, output_counter, reasoning, sequence)`

    4. **`delta.content` 处理（D-09: 首个 content 关闭 reasoning）：**
       - `content_text = delta.get("content")`
       - 若不为 None：
         - `current_output_type == "reasoning"` 时（D-09 transition）：
           - yield `_build_output_item_done(output_counter, "reasoning", item_id)`
           - `item_id_counter += 1` -> 新 text item
           - `output_counter += 1`
           - yield `_build_output_item_added(output_counter, "message", ...)`
           - yield `_build_content_part_added(item_id, output_counter, content_index=0)`
           - `current_output_type = "text"`
         - `current_output_type == "text"` 时维持（增量延续）：
           - 仅需 yield delta（已在当前 text item 内）
         - `current_output_type is None` 且首次 content 出现（纯文本无 reasoning）：
           - `item_id_counter += 1`
           - `output_counter += 1`
           - yield `_build_output_item_added(output_counter, "message", ...)`
           - yield `_build_content_part_added(item_id, output_counter, 0)`
           - `current_output_type = "text"`
         - `sequence += 1`
         - `accumulated_text += content_text`
         - yield `_build_output_text_delta(item_id, output_counter, content_text, sequence)`

    5. **`delta.tool_calls` 处理（D-07/D-08）：**
       - `tool_calls = delta.get("tool_calls")`
       - 若有：
         - 遍历 `tc in tool_calls`：
           - `idx = tc.get("index")`
           - 若 `idx not in active_tool_indices`（新工具调用，D-07）：
             - 若 `current_output_type == "text"`：先 yield text `output_item.done`
             - 提取工具信息：
               - `call_id = tc.get("id", f"call_{idx}")`
               - `name = tc.get("function", {}).get("name", "unknown")`
             - `active_tool_indices.add(idx)`
             - `tool_call_ids[idx] = call_id`
             - `tool_call_names[idx] = name`
             - `tool_call_arguments[idx] = ""`
             - `item_id_counter += 1` -> `f"fc_{item_id_counter}"`
             - `output_counter += 1`
             - yield `_build_output_item_added(output_counter, "function_call", name=name, call_id=call_id)`
             - `current_output_type = "tool_call"`
           - 收获 arguments 片段：
             - `arg_fragment = tc.get("function", {}).get("arguments", "")`
             - `tool_call_arguments[idx] += arg_fragment`
           - `sequence += 1`
           - yield `_build_function_call_arguments_delta(item_id, call_id, output_counter, arg_fragment, sequence)`

    6. **`finish_reason` 处理：**
       - 若 `finish_reason` 为真值（如 `"stop"`, `"length"`, `"tool_calls"`）：
         - 遍历 `active_tool_indices`（D-07 finish_reason 关闭所有）：
           - 对每个 idx：yield `_build_function_call_arguments_done(item_id, call_id, output_counter, tool_call_arguments[idx])`
           - yield `_build_output_item_done(output_counter, "function_call", item_id, name=tool_call_names[idx], call_id=tool_call_ids[idx], accumulated_arguments=tool_call_arguments[idx])`
         - 若 `current_output_type in ("reasoning", "text")`：
           - yield `_build_output_item_done(output_counter, current_output_type, item_id, accumulated_text=accumulated_text)` (或 reasoning 用对应 item_id)
         - 获取 usage: `chunk.get("usage")`
         - yield `_build_response_completed(response_id, usage)`
         - `_completed = True`
         - logger.info 记录完成信息

    7. **异常处理（mode=proxy.py L395 pattern）：**
       - 整个 `async for` 外层用 `try/except` 包裹
       - except: `logger.exception("[CODEX] SSE stream translation error")`
       - 若未 _completed，emit `response.completed` 作为优雅关闭

    **关键状态转换矩阵（供执行者参考）：**

    | 当前类型 | Delta 内容 | 操作 |
    |----------|------------|------|
    | None | reasoning_content | output_item.added(reasoning) + reasoning_text.delta |
    | None | content | output_item.added(message) + content_part.added + output_text.delta |
    | None | tool_calls[N] | output_item.added(function_call[N]) + function_call_arguments.delta |
    | reasoning | content | output_item.done(reasoning) + output_item.added(message) + content_part.added + output_text.delta |
    | reasoning | tool_calls[N] | output_item.done(reasoning) + output_item.added(function_call[N]) + function_call_arguments.delta |
    | text | reasoning_content | output_item.done(text) + output_item.added(reasoning) + reasoning_text.delta |
    | text | tool_calls[N] | output_item.done(text) + output_item.added(function_call[N]) + function_call_arguments.delta |
    | tool_call | reasoning_content | output_item.done(tool_call) + 遍历 active_tool_indices 发各自的 done + output_item.added(reasoning) |
    | tool_call | content | output_item.done(tool_call) + ...同步骤 + output_item.added(message) + content_part.added |
    | tool_call | tool_calls[N] (new index) | output_item.added(function_call[N]) + function_call_arguments.delta |
    | tool_call | tool_calls[N] (known index) | function_call_arguments.delta only |
    | any | finish_reason | 所有 active items 发 done + response.completed |
